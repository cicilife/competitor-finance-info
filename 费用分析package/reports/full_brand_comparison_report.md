## 八、指标计算方法与说明

### 8.1 核心指标定义

| 指标名称 | 计算公式 | 说明 |
|---------|---------|------|
| **费效比** | 营收 ÷ SG&A费用 | 每投入1元费用可产生多少营收，越高说明费用效率越好 |
| **营销费用率** | 营销费用 ÷ 营收 × 100% | 营销投入占营收比例，反映品牌建设力度 |
| **营销效率指数** | 营收增量 ÷ 营销费用增量 | 每增加1元营销费用带来多少营收增长，越高越好 |
| **毛利率** | (营收 - 成本) ÷ 营收 × 100% | 衡量核心盈利能力，不考虑费用 |
| **SG&A占比** | SG&A费用 ÷ 营收 × 100% | 费用占营收比例，越低说明费用控制越好 |
| **净利率** | 净利润 ÷ 营收 × 100% | 最终盈利水平 |
| **费用弹性** | 费用CAGR ÷ 营收CAGR | >1表示费用增长快于营收，<1表示效率改善 |

### 8.2 费效比判断标准

| 费效比范围 | 效率评级 | 说明 |
|-----------|---------|------|
| > 4.5 | ⭐⭐⭐⭐⭐ 卓越 | 费用控制极佳，运营效率行业领先 |
| 3.0 - 4.5 | ⭐⭐⭐⭐ 优秀 | 费用控制良好，效率高于行业平均 |
| 2.5 - 3.0 | ⭐⭐⭐ 中等 | 费用控制一般，行业平均水平 |
| 2.0 - 2.5 | ⭐⭐ 较差 | 费用控制较弱，需要改善 |
| < 2.0 | ⭐ 差 | 费用效率问题严重，急需整改 |

### 8.3 营销效率指标

**营销费用率基准：**
- 行业平均：10-12%
- 高端品牌：12-18%（需要更多品牌建设投入）
- 大众品牌：8-12%
- B2B主导品牌：5-8%（如捷安特）

**营销效率指数解读：**
- > 1.0：营销投入高效，每1元营销带来超过1元营收增长
- 0.5-1.0：营销效率中等
- < 0.5：营销效率较低

### 8.4 品牌定位矩阵（二维框架）

基于 **费用效率（费效比）** 和 **增长动能（营收CAGR）** 的组合：

```
                        高费用效率（费效比>3.0）
                               │
           ┌───────────────────┼───────────────────┐
           │                   │                   │
高增长     │   【成熟领导者】    │   【成长挑战者】   │
(CAGR>8%)  │  Nike, 安踏       │  On昂跑, 361度,  │
           │                   │  牧高笛, 特步,    │
           │                   │  Amer, 比音勒芬   │
           │                   │                   │
           ├───────────────────┼───────────────────┤
           │                   │                   │
低增长     │   【成熟但困境】    │   【新兴潜力】     │
(CAGR<8%)  │  VF集团, PUMA,    │  滔搏            │
           │  探路者           │                   │
           │                   │                   │
           └───────────────────┴───────────────────┘
                               │
                        低费用效率（费效比<3.0）
```

**象限说明：**
- **成熟领导者**：高费用效率+高增长 = 最佳状态
- **成长挑战者**：低费用效率+高增长 = 扩张期可接受
- **成熟但困境**：高费用效率+低增长 = 需要新增长引擎
- **新兴潜力**：低费用效率+低增长 = 需要谨慎评估

**多品牌优势分析：**

- **定位互补**：覆盖不同价格带和消费者群体，最大化市场覆盖
- **风险分散**：单一品牌下滑不影响整体盈利能力

**费用效率数据支撑：**

- 安踏费效比4.66，在21个品牌中**排名第二**（仅次于Nike）
