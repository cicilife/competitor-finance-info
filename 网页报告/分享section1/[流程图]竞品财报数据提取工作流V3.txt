                  阶段 4: 归一化处理 ⚠️ 关键
═══════════════════════════════════════════════════════════════════

                 │
                 ▼
        ┌──────────────────────────────────────┐
        │  11. 单位换算公式 (核心!)              │
        │                                      │
        │  ┌────────────────────────────────┐  │
        │  │ 百万美元:  × 1,000 × 7.2     │  │
        │  │ 千美元:    × 1     × 7.2     │  │
        │  │ 百万欧元:  × 1,000 × 7.8     │  │
        │  │ 10亿日元:  × 1,000,000 × 0.048│  │
        │  │ 百万日元:  × 1,000 × 0.048   │  │
        │  │ 千加元:    × 1     × 5.5     │  │
        │  │ 千瑞郎:    × 1     × 8.1     │  │
        │  │ 千元CNY:   × 1     × 1       │  │
        │  └────────────────────────────────┘  │
        │                                      │
        │  M列 (归一化指标数值) = G列 × 换算系数 │
        └────────┬─────────────────────────────┘
                 │
                 ▼
        ┌──────────────────────────────────────┐
        │  12. 衍生指标计算                      │
        │     - 净利率 = 净利润 / 营业收入      │
        │     - 毛利率 = 毛利 / 营业收入        │
        │     - 经营利润率 = 经营利润 / 营收    │
        │     - 运营费用率 = 运营费用 / 营业收入 │
        │     - 单店收入 = 营业收入 / 门店数    │
        │     - 库存周转天数 = 365 / 存货周转率 │
        └────────┬─────────────────────────────┘

═══════════════════════════════════════════════════════════════════
                    ⚠️ 核心避坑清单
═══════════════════════════════════════════════════════════════════

  1. 百万日元 vs 10亿日元 千万不能混 (×1000 vs ×1,000,000)
  2. G列原始值和M列归一化值不能相等 (检查单位换算)
  3. 区域优先级: 中国内地 > 大中华区 > 亚太区 > TOTAL
  4. 财年结束日期要标注 (FY20240331 vs FY20241231)
  5. 港股用大中华区, 美股用Greater China, 欧股用EMEA
  6. 家族企业 (迪卡侬) 披露有限, 只能用公开数据
  7. 季报数据×4/3 年度化 (美津浓特殊处理)
