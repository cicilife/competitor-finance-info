## 数据口径
- 数据源: 23 份原始年报 + 财务数据库 v3.1（标注日期 2026-06-07）
- 抽取方式: Python 爬虫 + 人工兜底
- 标注规则: 双重验证（费用类型 = 类型词 ∩ 费用词）
- 财年口径: FY2023 → FY2025 三大财年
  港股 / 美股 / 欧股（采用各自公司财年定义）
- 排名口径: 按 FY2024 净利率排名（不是按毛利率）
