## 📊 功能特点

### 多数据源支持
1. **官方IR** - 优先级最高，直接从公司官网获取
2. **财经平台** - 巨潮资讯、东方财富等
3. **搜索引擎** - 百度、必应（备用）

### 智能存储
- 按品牌分子文件夹
- 文件命名规则：`{品牌}_{年份}_{报告期}.pdf`
- 自动去重（已存在文件跳过）
- PDF有效性验证

### 容错机制
- SSL证书验证跳过
- 请求重试机制
- 异常捕获和日志记录

## 🚀 使用方法

### 基本命令

```bash

# 检查已下载
python tasks/pdf_crawler/main.py --check

# 爬取单个品牌
python tasks/pdf_crawler/main.py --brand 安踏体育

# 爬取所有品牌
python tasks/pdf_crawler/main.py
```

## ⚠️ 已知限制

### 官方IR网站
部分官方IR网站存在以下限制：
1. **需要JavaScript渲染** - 部分页面是动态加载
2. **需要登录** - 有些需要投资者账号
3. **反爬措施** - 部分网站有人机验证

### 解决方案
1. **优先使用财经平台** - 巨潮资讯等更易获取
2. **手动补充** - 对于爬取失败的，可以手动下载后放入对应文件夹

## 💡 手动获取财报的建议

由于部分官方IR网站有限制，建议：

### 中国公司（港股）
1. 打开 **巨潮资讯** (cninfo.com.cn)
2. 搜索公司名称
3. 查找"年度报告"、"季度报告"
4. 下载PDF

### 国际公司
1. 打开公司官方IR网站
2. 查找"Annual Report"、"Quarterly Results"
3. 下载PDF

### 存放位置
```
竞品财务PDF库/{品牌名}/{品牌名}_{年份}_{报告期}.pdf
```
