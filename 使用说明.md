# 竞品资讯系统使用说明

## 快速开始

### 安装依赖
```bash
pip install -r requirements.txt
```

### 初始化数据库
```bash
python main.py --init
```

### 提取财务数据
```bash
python main.py --extract-financial
```

---

## 主要功能

### 1. 财务数据提取

**特点**：
- 自动从PDF财报中提取关键指标
- 货币自动转换为人民币
- 数据质量自动验证
- Excel导出

**输出文件**：
- `data/processed/financial_data_YYYYMMDD.xlsx` - 统一管理的财务数据表

**提取的指标**：
- 营业收入(亿元)
- 净利润(亿元)
- 毛利率(%)
- 净利率(%)
- 门店数量
- 员工人数
- 等...

### 2. 新闻简报

**特点**：
- 自动从RSS源抓取竞品新闻
- 按品牌和内容类型分类
- 重点事件自动识别和提示

**输出文件**：
- `output/news_brief_YYYYMMDD.md` - 新闻简报

### 3. 定时任务

```bash
# 启动定时任务（每周一自动更新）
python main.py --schedule
```

## 问题分析与改进建议

### 当前问题
1. **品牌数据不足**：大部分品牌没有PDF财报
2. **提取准确率有限**：自动提取的数据准确性有待提升
3. **缺少历史数据**：只覆盖了1-2年的数据

### 短期改进建议（可立即实施）

创建一个Excel模板，手工录入：

- 李宁：2020-2025年财报数据
- 特步：2020-2025年财报数据
- 等主要品牌...

**优点**：数据质量高，可控性强

#### 方案B：增强爬虫功能
- 从新浪财经、雪球等平台抓取
- 抓取HTML格式的财报
- 利用现成的API数据

**优点**：自动化程度高

#### 方案C：混合方案
- 手工录入历史数据（2020-2025）
- 爬虫抓取最新数据（2026以后）
- 两者结合使用

**推荐方案**

## Excel输出说明

### 列说明
| 列名 | 说明 |
|------|------|
| 品牌名称 | 中文品牌名 |
| 英文名称 | 英文品牌名 |
| 报告类型 | 年报/季报 |
| 报告期 | 年份/季度 |
| 营业收入(亿元) | 统一货币为人民币 |
| 数据质量标记 | 良好/异常原因说明 |

### 数据质量标记
- **良好**：数据通过验证
- **缺失指标**：缺少关键数据
- **年份异常**：年份不在合理范围
- **数值异常**：数值超出合理范围

## 配置文件说明

### config/brands.yaml
配置品牌信息和数据源

### config/sources.yaml
配置新闻RSS源和搜索关键词

### config/settings.yaml
系统配置（汇率、更新时间等）
